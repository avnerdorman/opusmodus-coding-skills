degree-to-radian n
[Function]

Arguments and Values:
n                       a number or a list of numbers.


Description:

This function converts degrees to radians angle.
1º = π/180º = 0.005555556π = 0.01745329252 rad

(degree-to-radian 1)
=> 0.017453


Examples:
(degree-to-radian '(5 35 90 71))
=> (0.087266 0.610865 1.570796 1.239184)
