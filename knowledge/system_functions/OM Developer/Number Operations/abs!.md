Opusmodus                                             abs!   1

abs! n

[Function]

Arguments and Values:
n                      a number or list of numbers.


Description:

ABS! returns the absolute value of numbers.


Examples:
(abs! -1)
=> 1

(abs! '(0 -1 2 -3))
=> (0 1 2 3)

(abs! '((0 -1) (2 -3)))
=> ((0 1) (2 3))
