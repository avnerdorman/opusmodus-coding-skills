Opusmodus                                             neg!   1

neg! n

[Function]

Arguments and Values:
n                      a number or list of numbers.


Description:

NEG! returns the negative value of numbers.


Examples:
(neg! 1)
=> -1

(neg! '(0 -1 2 -3))
=> (0 -1 -2 -3)

(neg! '((0 -1) (2 -3)))
=> ((0 -1) (-2 -3))
